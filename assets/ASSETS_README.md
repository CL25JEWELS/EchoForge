# Assets Directory

This directory contains static assets for the Looper app.

## Required Assets

### App Icons
- `icon.png` - App icon (1024x1024px)
- `adaptive-icon.png` - Android adaptive icon (1024x1024px)
- `favicon.png` - Web favicon (48x48px)

### Splash Screen
- `splash.png` - Splash screen (1242x2436px for iPhone X)

## Creating Assets

You can create these assets using:
1. Design tools (Figma, Adobe XD, Sketch)
2. Online generators (https://www.appicon.co/)
3. Expo asset generator: `npx expo-optimize`

## Asset Guidelines

### Icon Design
- Use a simple, recognizable symbol
- High contrast for visibility
- Works well at small sizes
- Consistent with app branding

### Color Scheme
- Primary: #6c5ce7 (purple)
- Secondary: #fd79a8 (pink)
- Accent: #00d2d3 (cyan)
- Background: #1a1a2e (dark blue)

### Suggested Icon Concept
A stylized loop/circle with musical note or grid pattern representing the loop pad.

## Temporary Placeholders

Until custom assets are created, Expo will use default placeholders.
The app will still function normally, but custom icons are recommended for production.
